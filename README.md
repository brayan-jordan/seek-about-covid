## Setup da aplicação localmente

1. Clone o repositório
    ```bash
    git clone https://github.com/brayan-jordan/seek-about-covid.git
    ```
2. Instale as dependências 
    ```bash
    npm install
    ```
3. Execute o projeto
    ```bash
    npm start
    ```
4. Acesse o projeto em http://localhost:3000
